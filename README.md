## 📌 Project Overview

This is my personal developer portfolio — a fully handcrafted website that showcases my skills, projects, experience, and education. The design intentionally breaks away from the generic dark-purple developer portfolio trend.

The aesthetic is inspired by **editorial print design** — warm paper tones, bold ink-style typography, newspaper-like grid layouts, and subtle film grain texture. Every section is carefully composed to feel like a designed artifact, not just a webpage.

Built with **zero frameworks** — pure HTML, CSS, and vanilla JavaScript.

## ✨ Features

- **Editorial Design System** — Warm cream (`#f5f0e8`) paper background with rust red (`#c94a2a`) as the signature accent color
- **Custom Cursor** — Smooth rust-dot cursor with a trailing ring that expands on hover
- **Scroll Reveal Animations** — Staggered entrance animations using IntersectionObserver API
- **Infinite Marquee Strip** — Continuously scrolling tech skills banner between sections
- **Split Hero Layout** — Two-column hero with editorial photo treatment (grayscale + warm overlay)
- **Hover Interactions** — Skill cells reveal large ghost watermark letters; project rows shift on hover
- **Noise Texture Overlay** — SVG-based grain texture gives a printed paper feel
- **Responsive Design** — Fully mobile-friendly, collapses gracefully at 900px breakpoint
- **Zero Dependencies** — No frameworks, no libraries, no build tools needed
- **Fast Load** — Lightweight pure HTML/CSS/JS, hosted on Netlify CDN
